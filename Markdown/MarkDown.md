# MarkDown语法的应用 
----
### 一、MarkDown的认识
- 一种轻量的、易于书写的、纯文本格式的语法，用来在互联网上书写带有格式的文档。

- MarkDown 用来控制文档的样式，包括：文字的粗体、斜体，在文档中添加图片，创建列表，等等。

- MarkDown 是由 John Gruber 与 Aaron Swartz 在 2004 年合作发明的，距今已经有十多年的历史了。
